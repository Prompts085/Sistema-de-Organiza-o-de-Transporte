# Sistema de Organização de Transporte

## Como baixar o projeto/trabalho:

git clone https://github.com/Prompts085/Sistema-de-Organiza-o-de-Transporte.git

## Tecnologia utilizada:
- Python

## Sobre o projeto/trabalho:
- Esse projeto foi desenvolvido com um proposito de auxiliar a organização e a
segurança dos alunos no onibus durante o trajeto, garantindo que todos vão sentados.

## Funcionalidades:
- Temos o cadastro de alunos, onde vai ser escolhido dentro de um dicionário aninhado que tem as rotas e o turnos (matutino ou vespertino).

## Controle de Vagas:
- As vagas são subtraídas em tempo real de um dicionário aninhado que separa turnos matutino e vespertino.
